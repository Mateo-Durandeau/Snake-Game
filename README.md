# Snake Game

Ce projet est une implémentation moderne du jeu classique Snake, développé en C++ en utilisant la bibliothèque SFML. Le jeu inclut des fonctionnalitées comme des couleur aléatoire pour le serpent à chaque mort. Un compteur du meilleur score. Ou bien un démarrage qui se lance que quand on presse un bouton du clavier. 


## Prérequis

Pour compiler et exécuter ce jeu, vous aurez besoin de :

- Un compilateur C++ prenant en charge C++11 ou ultérieur (comme GCC, Clang, ou MSVC).
- La bibliothèque SFML installée sur votre système. Consultez [la documentation officielle SFML](https://www.sfml-dev.org/tutorials/2.5/) pour les instructions d'installation.

## Compilation et Lancement

### Sous Windows

1. Ouvrez votre projet dans Visual Studio.
2. Configurez les chemins d'inclusion et de bibliothèque pour SFML dans les propriétés du projet.
3. Sélectionnez `Build -> Build Solution` pour compiler le jeu.
4. Exécutez le fichier `.exe` généré depuis le dossier `Debug` ou `Release`.
